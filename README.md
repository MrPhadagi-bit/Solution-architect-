# Solution Architect Project Documentation

This repository is structured around the simple documentation style in the reference repo, but expanded so each project has its own folder for:

- `README.md` for the Markdown write-up
- `pdf/` for the project manual PDF
- `images/` for screenshots used in the write-up

The project list below keeps the exact order you provided. The duplicate `Cloud Security with AWS IAM` entry appears twice on purpose as projects `02` and `16`.

## Repository Layout

```text
projects/
  01-host-a-website-on-amazon-s3/
    README.md
    pdf/
    images/
  ...
templates/
  project-template.md
```
